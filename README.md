# Prasart Jongjaroenkamol personal website

This is a simple static website for GitHub Pages.

## Files

- `index.html` contains the website content.
- `styles.css` controls the visual design.

## Publishing on GitHub Pages

Create a public GitHub repository named:

```text
prasart.github.io
```

Upload `index.html`, `styles.css`, and this `README.md` to the repository.
GitHub Pages will publish the site at:

```text
https://prasart.github.io
```

## Updating later

Most future updates can be made by editing text in `index.html`.
